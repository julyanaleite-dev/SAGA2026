# Keynote — Saga SENAI de Inovação · Capítulo I

Apresentação HTML offline criada para a palestra **“A história que nos trouxe até aqui”**.

## Arquivo principal

Abra o arquivo:

`keynote-capitulo-i/index.html`

## Como usar

1. Baixe o repositório ou o arquivo `index.html`.
2. Abra no navegador, de preferência Google Chrome ou Microsoft Edge.
3. Use as teclas:
   - `←` e `→` para navegar;
   - `F` para tela cheia;
   - `O` para visão geral;
   - `P` para imprimir/exportar em PDF.

## Características

- 30 slides.
- HTML single-file, sem dependência de internet.
- Navegação inspirada no Reveal.js.
- Animações CSS/JS.
- Identidade visual SENAI.
- Ilustrações vetoriais inline inspiradas nos conceitos enviados: prototipagem, fala em público, equipe, orçamento/valor, conferência, jornada de aprendizagem, rede, comunicação e celebração.

## Observação

A estrutura não antecipa a nova Saga 2026. Ela funciona como **Capítulo I**, preparando o público para o segundo momento do evento.
