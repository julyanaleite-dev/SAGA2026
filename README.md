## Como publicar no GitHub Pages

1. Acesse **Settings** do repositório.
2. Vá em **Pages**.
3. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
4. Salve.

Após a publicação, o site deverá ficar disponível em:

`https://julyanaleite-dev.github.io/SAGA2026/`

## Como navegar

- Setas `←` e `→`: navegar entre slides.
- `F`: tela cheia.
- `O`: visão geral.
- `R`: alternar modo site contínuo.
- `P`: imprimir ou exportar em PDF.

## Diretriz visual

A apresentação foi redesenhada como um site vivo, com:

- HTML, CSS e JS separados;
- assets versionados;
- ilustrações SVG no repositório;
- design responsivo;
- navegação por teclado;
- modo apresentação e modo site contínuo;
- identidade visual SENAI com azul institucional, azul de apoio e laranja de destaque.
